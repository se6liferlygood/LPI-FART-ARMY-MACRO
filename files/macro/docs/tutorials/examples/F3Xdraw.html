you can draw with F3X using a forever loop and a statement

LF KJ C W17 (‡K‡) END

LF and END is loop, K is key input, C is click, W is wait, (‡K‡) is a statement
you place out a part with J in f3x
C clicks to place out part
W17 waits for 17ms ≈ (1000/60)ms which leads to ~60 KJ and C per second
(‡K‡) makes sure END only happens while your holding the keybind

this forever loop only runs while your holding the keybind
you can use this to draw with f3x
