lets imagine that we have 5 boxes stacked next to each other like this
(it could be any amount of boxes)

index 1
index 2
index 3
index 4
index 5

we only know the x axis value for box 1 and 5
all of the boxes share the same y value

lets call the x value of box 1 x1
lets call the x value of box 5 x2
they share the same y value

heres what I mean

index 1
(x1,y)
index 2
index 3
index 4
index 5
(x2,y)

so from this I want to make a formula which gives the x value for any box
this formula will only use these 2 x values

the formula goes like this:

n: amount of boxes
i: box index

we store the a value
a = (x2-x1)/(n-1)

and use it in this formula which gives the x axis value for any box index:
(i-1)*a+x1

so for example if x1 is 1 and x2 is 5 and theres 5 boxes
then a would be: (5-1)/(5-1) = 1
we store the a value and use that in our formula
now lets say I want the x value for box number 3
so I plug in 3 into the formula by replacing i and get:
(3-1)*1 + 1 = 3
this makes sense

heres how I came up with this formula
what I want is a function which gives the x axis value for any box
using the x value of the first and last box
it is obvious that this function must be linear so I did this:

f(x) = a*x+b
f(1) = x1
f(n) = x2

solve for a and b given these 3 equations
the solution is obvious but I will write it out without using any formulas

we write f(1) and f(n) as this using f(x):
f(1) = a+b = x1
f(n) = a*n+b = x2

now lets focus on these 2 equations
1: a+b = x1
2: a*n+b = x2

we can see this from the first equation:
b = x1-a

subtract the 2nd equation by the first equation and we get this this:
x2-x1 = a*n+b-(a+b)
x2-x1 = a*n-a
x2-x1 = a*(n-1)

divide by (n-1) and we get this:
a = (x2-x1)/(n-1)

so now we know:
a = (x2-x1)/(n-1)
b = x1-a

we substitue for b in f(x) and we get this:
f(x) = a*x+(x1-a)
f(x) = (x-1)*a+x1

so the only thing we need is to calculate a and store it in a variable
then use that for the formula in our macros
a = (x2-x1)/(n-1)
f(x) = (x-1)*a+x1

you can also come up with this formula by just looking at the boxes
