inbuilt macros allows you to use fart army macros and other functions in your scripts

heres a list of inbuilt macros:
CS: lag switch on (clientside)
SS: lag switch off (serverside)
SR: server rejoin
SH: serverhop
F1: freeze
F0: unfreeze

when you use CS the script will automatically stop lag switching when the script ends for safety reasons
also when you use F1 the script will automatically do unfreeze when it ends for the same reason
