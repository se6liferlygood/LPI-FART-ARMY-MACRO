hold & release keys

go back to main page

you can hold keys using H
for example:

HWD

would hold the keys W and D
but it would only be held for as long as the script is running or until they are released by R
to notice the keys being held you should use wait

you can release keys by using R
for example:

HW W1000 RW HD W1000

W is wait
would hold W for 1 second then release W then hold D for 1 second!

IF YOU ONLY TYPE R THEN IT RELEASES ALL HELD DOWN KEYS BY THE SCRIPT!
