you use special keys with {}
for example:

{ENTER}

would press enter
another example:

{SPACE DOWN} W1000 {SPACE UP}

W is wait
would hold space for 1 second
if you forget UP in special keys then it stays down until press that key

you cant jump in roblox if you press space in an instant
space must be held down for a certain amount of time for jump to happen
you can use DOWN and UP with special keys input!
