KEY LOGIC

the inbuilt variable ‡K‡ is very usefull here

key logic is used so that actions only happens if keys are held a certain way
keys you type out will be true if they are being held
theres 2 ways to type out keys
for single letters you can just type them out
for number keys or other keys you use {}, special keys
for example:

W1000 (W) K1

W is wait, K is key input
would only press 1 if W is being held
another example using the inbuilt variable ‡K‡

W1000 (‡K‡) K1

this script would only press 1 if if the script keybind key is being held
for example if the keybind is ALT A then 1 would only be pressed if A is being held
theres another example of ‡K‡ being used in variables to make an autoclicker

you can specify keys with {}, it is the same keys as special keys but you cant do up or down then
you also use {} to specify numbers keys
heres an example using number key and space:

WF1000 ({1}) T(you are holding 1!) ({SPACE}) T(you are holding space!)

WF is wait flag, T is text input
this script would wait 1 second then check if you are holding 1
if you are holding 1 it will type out: "you are holding 1!"
then it will wait 1 second then check if your holding space
if you are holding space then it will type out: "you are holding space!"

examples using && (AND) and || (OR)

W1000 (Q&&E) K1

this script only only press 1 if Q and E are being held

W1000 (Q||E) K1

this script would only press 1 if Q or E or both are being held

lets say you wanna pause a script if any WASD is being held
you can do that using || (OR) and ! (NOT):

LF !(W||A||S||D) L C K21 END W100 !(‡K‡) END

LF, L and END is loop, C is click
using this fact about loops and statements
we can make sure that parts of script is only running if a statement is true
C K21 only happens if W or A or S or D is NOT being held!
this means NOTHING except wait happens if any WASD is being held
!(‡K‡) before the last END makes sure that the LF only loops if the script keybind is NOT being held!
this means for example if the keybind was ALT A then if you were holding A the script would stop!
