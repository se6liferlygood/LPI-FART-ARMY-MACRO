you do loops using L and END
for example:

L10 K1 W10 END

W is wait, K is key input
would press the key 1 10 times

it is the code between L and END that is executed as many times as you put after L

now lets say you wanna loop forever or until you stop it!
you can loop forever using LF
heres an example:

LF K1 W1000 END

this could for example be an AFK script making sure you dont get kicked
it presses the key 1 every second

but how do you stop forever loops or big loops?
you can do that by pressing CTRL ALT E

there is another way to stop forever loops though
you can do that using statements

now how do you get what loop count you are on?
you can do that using the inbuilt variable: ‡I‡
it starts at 1 and ends at the number you chose after L, with LF it keeps on going forever
for example:

WF25 L9 K‡I‡ C END K0 C

WF is wait flag, C is click
this script would press the keys 1 to 9 and then 0
it would click each time it presses a key
this script would use your entire toolbar in roblox
