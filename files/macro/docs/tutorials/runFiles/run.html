you can run files by using F and FW
F runs the file
FW pauses the macro until the file you ran has closed

the files you run are located in files\macro\custom\

for example lets say I have a file named helloWorld.bat in the custom folder
I would run it like this with F:

F(helloWorld.bat)

with FW:

FW(helloWorld.bat)
