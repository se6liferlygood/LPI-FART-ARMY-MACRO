a variable is typed with this notation: ‡X‡
you press ALT+SPACE to type it with fart army macro

a variable is something in which a value is stored
it can be a statement result, number or text in fart army script

INBUILT VARIABLES

here is a list of inbuilt variables:
mouse x: ‡X‡
mouse y: ‡Y‡
screen width: ‡W‡
screen height: ‡H‡
current loop count: ‡I‡
current script keybind: ‡K‡
time since boot in miliseconds: ‡T‡

the inbuilt variable ‡K‡ tells what key your keybind is
for example if a script used ALT K keybind then ‡K‡ would be K
heres an example of ‡K‡ being used:

LF C W10 (‡K‡) END

LF & END is loop, C is click, W is wait, (‡K‡) is a statement
this script would be autoclicking for as long as you hold the keybind

the inbuilt variable ‡T‡ can be used to measure time in miliseconds (delta time)
heres an example of delta time:

V‡time‡=‡T‡ W100 V‡dTime‡=‡T‡-‡time‡ T(‡dTime‡)

T() is text input, V is custom variable
this script would type out a number close to 100

CUSTOM VARIABLES

you can make your own variables with V
it works like this: V‡t‡=1
here I set the variable ‡t‡ to 1
this means K‡t‡ would press 1
you can do this is as well: V‡t‡=‡t‡+1
then K‡t‡ would type out 2 since it added 1 to 1
heres an example of variable being used:

V‡c‡=0 L10 V‡c‡=‡c‡+1 END T(counted up to ‡count‡)

L10 & END is loop, T() is text input
this script would add 1 to the variable ‡c‡ 10 times
it types out: counted up to 10

you can also store text in variables with ""
it works like this:

V‡text‡="hello world!" T(‡text‡)

this script would type out: hello world!
it is good to use variables to avoid copy pasting big things over and over again
they can also be used to calculate needed values
