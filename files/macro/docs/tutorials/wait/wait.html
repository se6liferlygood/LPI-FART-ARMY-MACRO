you do wait using W
wait works in miliseconds
1 second is 1000 miliseconds
for example:

K1 W1000 K2

K is key input
would press 1, wait 1 second then press 2

but it can be annoying to type many W
thats why wait flag exists, WF
wait flag adds a wait before every input action in miliseconds
heres an example where wait flag is used:

K1 WF500 K2 K3

this would press the key 1, wait 500ms then press 2, wait 500ms then press 3

you can use WF0 to disable wait flag if you have used it before

wait flag only happens for input actions and wait flag does not happen if W is used before
this means W0 can be used to skip wait flag between input actions
